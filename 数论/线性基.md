# 线性基

线性基用于处理异或相关的优化问题，可以找出数组中能异或出的最大值、最小值等。

```cpp
vector<int> basis(61, 0);

void insert(int x) {
    for (int i = 60; i >= 0; i--) {
        if (!(x & (1LL << i))) continue;
        if (!basis[i]) {
            basis[i] = x;
            return;
        }
        x ^= basis[i];
    }
}
```

## 使用说明

- `basis[i]` 表示第 i 位的基
- `insert(x)` 将数字 x 插入线性基
- 可以查询异或最大值：从高位到低位贪心选择
