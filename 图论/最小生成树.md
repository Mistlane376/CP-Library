# 最小生成树

## Prim算法

贪心算法，针对 $n-1$ 条边。

```cpp
vector<pair<int, int>> p[n + 1];  // 邻接表 (终点, 权值)
vector<int> vis(n + 1, 0);
vector<long long> dis(n + 1, LLONG_MAX);

dis[1] = 0;

for (int i = 1; i < n; i++) {  // 循环n-1次
    int cur = -1;
    long long minnum = LLONG_MAX;

    for (int j = 1; j <= n; j++) {
        if (!vis[j] && minnum > dis[j]) {
            cur = j;
            minnum = dis[j];
        }
    }

    // if (cur == -1) 图不连通

    vis[cur] = 1;

    for (auto [x, t] : p[cur]) {
        if (!vis[x] && t < dis[x]) {
            dis[x] = t;
        }
    }
}

// dis数组存储的即为最小生成树的边权值
// 求和即为最小生成树的权值和
```

**算法说明**：
- 从任意节点开始（这里是节点 1）
- 每次选择距离集合最近的未访问节点
- 更新相邻节点的距离
- 时间复杂度：$O(n^2)$，可以使用优先队列优化到 $O(E \log V)$

## Kruskal算法

按边贪心，用**并查集**维护连通性。

```cpp
struct Edge {
    int u, v, w;
    bool operator<(const Edge &other) const {
        return w < other.w;
    }
};

vector<Edge> edges;

int ans = 0, cnt = 0;
sort(edges.begin(), edges.end());

for (auto &e : edges) {
    int u = e.u, v = e.v, w = e.w;
    if (unio(u, v)) {  // 并查集合并
        ans += w;
        cnt++;
        if (cnt == n - 1) break;
    }
}

if (cnt != n - 1) ans = -1;  // 图不连通
```

**对比 Prim vs Kruskal**：
- Prim：适合稠密图
- Kruskal：适合稀疏图

## 并查集模板

```cpp
vector<int> par(n + 1), sz(n + 1);

void init(int n) {
    for (int i = 1; i <= n; i++) {
        par[i] = i;
        sz[i] = 1;
    }
}

int find(int x) {
    if (par[x] == x) return x;
    return par[x] = find(par[x]);  // 路径压缩
}

bool unio(int x, int y) {
    x = find(x);
    y = find(y);
    if (x == y) return false;

    // 按大小合并
    if (sz[x] < sz[y]) swap(x, y);
    par[y] = x;
    sz[x] += sz[y];
    return true;
}
```
