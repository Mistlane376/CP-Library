# 树状数组（二叉索引树）

```cpp
template<class T>
struct BIT {
    int n;
    vector<T> a;

    BIT(int sz = 0) : n(sz), a(sz + 1) {}

    void add(int p, T v) {
        for (; p <= n; p += p & -p) a[p] += v;
    }

    T ask(int p) {
        T r = 0;
        for (; p; p -= p & -p) r += a[p];
        return r;
    }
};
```

**算法说明**：
- `add(p, v)`：在位置 p 增加 v
- `ask(p)`：查询前缀和 [1, p]
- 时间复杂度：O(log n)

**区间求和**：
```cpp
T query(int l, int r) {
    return ask(r) - ask(l - 1);
}
```

**特点**：
- 实现简单
- 空间效率高
- 适合前缀和、区间更新等场景
