# 背包问题

## 多重背包 - 二进制优化

```cpp
int amount = s[i];
for (int k = 1; amount > 0; k <<= 1) {
    int num = min(k, amount);
    int volume = num * v[i];
    int value = num * w[i];

    for (int j = m; j >= volume; j--) {
        dp[j] = max(dp[j], dp[j - volume] + value);
    }
    amount -= num;
}
```

**算法说明**：
- `s[i]`：第 i 种物品的数量
- `v[i]`：第 i 种物品的体积
- `w[i]`：第 i 种物品的价值
- 使用二进制拆分，将多重背包转化为 01 背包
- 时间复杂度：O(m log s[i])

**为什么使用二进制？**
- 将数量 s 拆分为 1, 2, 4, 8, ..., 2^k
- 任意数量 [0, s] 都可以由这些数组合而成
- 物品总数控制在 O(log s) 级别

**其他背包模板**：
- 01 背包：倒序遍历 dp 数组
- 完全背包：正序遍历 dp 数组
- 分组背包：每组物品选一个
