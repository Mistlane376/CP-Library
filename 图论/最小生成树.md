# 最小生成树

## Prim算法

```cpp
vector<long long> dis(n + 1, LLONG_MAX);
dis[1] = 0;

for (int i = 1; i < n; i++) {
    int cur = -1;
    long long minnum = LLONG_MAX;
    for (int j = 1; j <= n; j++) {
        if (!vis[j] && minnum > dis[j]) {
            cur = j;
            minnum = dis[j];
        }
    }
    vis[cur] = 1;
    for (auto [x, t] : p[cur]) {
        if (!vis[x] && t < dis[x]) {
            dis[x] = t;
        }
    }
}
```

**算法说明**：
- 从任意节点开始（这里是节点 1）
- 每次选择距离集合最近的未访问节点
- 更新相邻节点的距离
- 时间复杂度：O(n²)，可以使用优先队列优化到 O(E log V)

## Kruskal算法

```cpp
struct Edge {
    int u, v, w;
    bool operator<(const Edge &other) const {
        return w < other.w;
    }
};
```

**补充**：Kruskal 需要并查集（DSU）支持，按边权从小到大排序，依次选择不形成环的边。

**对比 Prim vs Kruskal**：
- Prim：适合稠密图
- Kruskal：适合稀疏图
