# 常用函数

## STL算法

```cpp
next_permutation(a.begin(), a.end());  // 全排列生成
```

## 进制运算的内置函数（GCC/Clang）

```cpp
__builtin_clz(unsigned int x);  // 统计前导零的数量
__builtin_ctz(unsigned int x);  // 统计末尾零的数量
__builtin_popcount(unsigned int x);  // 统计1的数量
```

## C++ 匿名函数（Lambda 表达式）

```cpp
// 基本语法
[捕获列表](参数列表) -> 返回类型 { 函数体 }

// 示例
auto add = [](int a, int b) -> int { return a + b; };

// 递归lambda需要指定类型
function<返回类型(传入类型)> func = [&](参数) -> 返回类型 {
    // 函数体
};
```

## 其他常用函数

### 数学函数

- `abs(x)` - 绝对值
- `max(a, b)` / `min(a, b)` - 最大/最小值
- `swap(a, b)` - 交换两个变量
- `fill(a, a + n, val)` - 填充数组
- `lower_bound(first, last, val)` - 有序容器中第一个 >= val 的位置
- `upper_bound(first, last, val)` - 有序容器中第一个 > val 的位置
- `binary_search(first, last, val)` - 二分查找
- `sort(first, last)` - 排序（默认升序）
- `reverse(first, last)` - 反转
- `unique(first, last)` - 去重（返回去重后末尾的迭代器）

### 输入输出优化

```cpp
// 加速输入输出
ios::sync_with_stdio(false);
cin.tie(nullptr);

// 或使用 scanf/printf（速度更快）
scanf("%d", &x);
printf("%d\n", x);
```
