# 其他技巧

## `__int128` 的使用

```cpp
__int128 read() {
    char arr[30];
    __int128 res = 0;
    scanf("%s", arr);
    for (int i = 1; i <= strlen(arr); i++) {
        res = res * 10 + (arr[i] - '0');
    }
    return res;
}
```

**适用场景**：
- 处理超过 long long 范围的大整数
- 中间计算防止溢出
- 输入输出需要自定义函数

**注意**：`__int128` 是 GCC/Clang 扩展，不是标准 C++。

## 判断回文数

```cpp
bool isreverse(int x) {
    if (x < 0) return false;
    if (x != 0 && x % 10 == 0) return false;

    int reversedHalf = 0;
    while (x > reversedHalf) {
        reversedHalf = reversedHalf * 10 + x % 10;
        x /= 10;
    }

    return x == reversedHalf || x == reversedHalf / 10;
}
```

**优化点**：
- 只反转一半数字，避免溢出
- 负数直接返回 false
- 以 0 结尾的非零数字直接返回 false

## 差分

```cpp
// 区间修改：[l, r] 增加 k
// 差分数组 d
// d[l] += k; d[r + 1] -= k;

// 前缀和还原
for (int i = 1; i <= n; i++) {
    d[i] += d[i - 1];
}
```

**适用场景**：
- 多次区间修改
- 最后统一查询

## 前缀和

```cpp
vector<int> pre(n + 1, 0);
for (int i = 1; i <= n; i++) {
    pre[i] = pre[i - 1] + a[i];
}

// 区间求和 [l, r]
int sum = pre[r] - pre[l - 1];
```

**优势**：
- O(1) 查询区间和
- O(n) 预处理
