# 二分图与匹配

## 最大匹配（匈牙利算法）

```cpp
vector<int> vis(b.size()), match(b.size(), -1);
int t = 0;

function<bool(int)> dfs = [&](int u) -> bool {
    for (auto v : a[u]) {
        if (vis[v] == t) continue;
        vis[v] = t;
        if (match[v] == -1 || dfs(match[v])) {
            match[v] = u;
            return true;
        }
    }
    return false;
};
```

**使用说明**：
- `a[u]`：左边节点 u 的所有邻接点
- `b.size()`：右边节点数量
- `match[v] = u`：右边节点 v 匹配左边节点 u
- `t++`：每次DFS前增加 t 以重置访问标记

**时间复杂度**：O(VE)，其中 V 是顶点数，E 是边数
