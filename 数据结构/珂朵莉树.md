# 珂朵莉树（区间赋值操作）

珂朵莉树（Chtholly Tree / ODT）是一种基于有序集合的数据结构，适用于大量随机数据的区间赋值操作。

```cpp
struct Node {
    int l, r;
    mutable int v;

    Node(int L, int R = -1, int V = 0) : l(L), r(R), v(V) {}

    bool operator<(const Node &other) const {
        return l < other.l;
    }
};
```

**基本操作**：

```cpp
set<Node> s;

// 初始化
auto itr = s.lower_bound({l, l});
// 分裂操作
auto split(int pos) {
    auto it = s.lower_bound({pos, pos});
    if (it != s.end() && it->l == pos) return it;
    --it;
    int L = it->l, R = it->r, V = it->v;
    s.erase(it);
    s.insert({L, pos - 1, V});
    return s.insert({pos, R, V}).first;
}

// 区间赋值
void assign(int l, int r, int v) {
    auto itr = split(r + 1), itl = split(l);
    s.erase(itl, itr);
    s.insert({l, r, v});
}
```

**适用场景**：
- 区间赋值
- 区间查询
- 数据随机性较高时效果更佳

**复杂度**：期望 O(log n) 单次操作
