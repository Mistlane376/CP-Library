# 单调栈

单调栈用于维护单调性，常用于求解"下一个更大/更小元素"等问题。

```cpp
vector<int> st(n + 1);
int top = 0;
for (int i = 1; i <= n; i++) {
    while (top && a[i] > a[st[top]]) {
        ans[st[top]] = i;  // 记录下一个更大元素的位置
        top--;
    }
    st[++top] = i;
}
```

**常见应用**：

### 1. 下一个更大元素（NGE）

```cpp
vector<int> next_greater(n, -1);
for (int i = 0; i < n; i++) {
    while (!st.empty() && a[st.top()] < a[i]) {
        next_greater[st.top()] = i;
        st.pop();
    }
    st.push(i);
}
```

### 2. 单调递增栈

将 `a[i] > a[st[top()]]` 改为 `a[i] >= a[st[top()]]` 处理相等元素。

### 3. 柱状图中最大的矩形

```cpp
while (!st.empty() && a[i] < a[st.top()]) {
    int height = a[st.top()];
    st.pop();
    int width = st.empty() ? i : i - st.top() - 1;
    area = max(area, height * width);
}
st.push(i);
```

**时间复杂度**：O(n)，每个元素最多入栈和出栈一次
