# 其他数论算法

## 秦九韶算法

用于快速计算多项式的值：

```cpp
long long ans = 0;
for (int i = n; i >= 0; i--) {
    ans = (ans * x + a[i]) % mod;
}
```

**原理**：将多项式 $P(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_0$ 改写为
$P(x) = (...((a_n x + a_{n-1}) x + a_{n-2}) ...) x + a_0$

## 扩展欧几里得算法

```cpp
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b, x, y);
    ll t = x;
    x = y;
    y = t - (a / b) * y;
    return d;
}
```

**用途**：
- 求 GCD
- 求乘法逆元（当 a 和 mod 互质时）
- 求解线性同余方程
