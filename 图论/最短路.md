# 最短路

## Floyd-Warshall（多源最短路）

```cpp
for (int k = 0; k < n; k++) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
        }
    }
}
```

**适用场景**：
- 求所有点对之间的最短路
- 处理负权边（无负环）
- 点数和边数较少时使用（复杂度 O(n³)）
