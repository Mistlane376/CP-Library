# 背包问题

## 多重背包 - 二进制优化

时间复杂度：$O(T \times m \times \sum \log(s_i))$

```cpp
int amount = s[i];
for (int k = 1; amount > 0; k <<= 1) {
    int num = min(k, amount);
    int volume = num * v[i];
    int value = num * w[i];

    // 01背包
    for (int j = m; j >= volume; j--) {
        dp[j] = max(dp[j], dp[j - volume] + value);
    }

    amount -= num;
}
```

**算法说明**：
- `s[i]`：第 i 种物品的数量
- `v[i]`：第 i 种物品的体积
- `w[i]`：第 i 种物品的价值
- 使用二进制拆分，将多重背包转化为 01 背包
- 时间复杂度：O(m log s[i])

**为什么使用二进制？**
- 将数量 s 拆分为 1, 2, 4, 8, ..., 2^k
- 任意数量 [0, s] 都可以由这些数组合而成
- 物品总数控制在 O(log s) 级别

## 多重背包 - 单调队列优化

```cpp
const int maxn = 1e6 + 10;
vector<int> dp(maxn), backdp(maxn);

for (int i = 1; i <= n; i++) {
    backdp = dp;
    int volume = v[i];
    int value = w[i];
    int amount = s[i];

    // 分为 0~volume 组，r表示余数
    for (int r = 0; r < volume; r++) {
        deque<int> dq;

        for (int k = 0; r + k * volume <= m; k++) {
            int j = r + k * volume;

            // dp[j] = max(backdp[t] + (k-t) * value)
            // 即 dp[j] > backdp[t] - t*value + k*value

            while (!dq.empty() &&
                   backdp[j] - k * value >= backdp[dq.back()] -
                   (dq.back() - r) / volume * value) {
                dq.pop_back();
            }

            dq.push_back(j);

            // 移除超出数量限制的元素
            while (!dq.empty() && (j - dq.front()) / volume > amount) {
                dq.pop_front();
            }

            // 更新dp[j]
            dp[j] = max(dp[j],
                        backdp[dq.front()] + (j - dq.front()) / volume * value);
        }
    }
}
```

**其他背包模板**：
- 01 背包：倒序遍历 dp 数组
- 完全背包：正序遍历 dp 数组
- 分组背包：每组物品选一个
