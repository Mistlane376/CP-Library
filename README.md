## ⚠️ 常见问题

### 模板使用方法

1. **直接复制**：点击对应模板即可复制代码
2. **了解背景**：每个模板都附有算法说明和适用场景
3. **注意细节**：部分模板需要根据题目调整参数（如数组大小、模数等）

### 注意事项

1. 计算连续段的初始长度时，条件不符时应初始化为 1。
2. `map` 和 `unordered_map` 的性能差异需要注意，尽量选择合适的数据结构。
3. 变量命名尽量避免冲突。
4. 注意运算符优先级，复杂表达式最好加括号。
5. 取模运算时要特别注意细节。
6. BFS/DFS 的标记最好在入队前完成。
7. 遇到环、多重连边或删除元素时，一般要换成链表或图论思维去处理。
8. 手动模拟是发现逻辑错误的重要方法。
