# 组合数

## 阶乘预处理

```cpp
vector<int> fac(n + 1, 1);
for (int i = 1; i <= n; i++) {
    fac[i] = (long long)fac[i - 1] * i % mod;
}

auto C = [&](int a, int b) -> int {
    if (a < 0 || b < 0 || b > a) return 0;
    return (long long)fac[a] * inv(fac[b]) % mod * inv(fac[a - b]) % mod;
};
```

**注意**：使用前需要先预处理阶乘数组 `fac`，并确保模数 `mod` 为质数。
