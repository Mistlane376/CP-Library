# 快速幂与逆元

## 快速幂模板

```cpp
int qpow(int base, int exp, int mod) {
    int res = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp & 1) res = (long long)res * base % mod;
        base = (long long)base * base % mod;
        exp >>= 1;
    }
    return res;
}
```

## 乘法逆元

```cpp
// 逆元（mod为质数时）
int inv(int n, int mod) {
    return qpow(n, mod - 2, mod);
}

// a * inv(b) % mod 等价于 a / b % mod
```

## 扩展欧几里得求逆元（mod为非质数）

```cpp
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    ll d = exgcd(b, a % b, x, y);
    ll t = x;
    x = y;
    y = t - (a / b) * y;
    return d;
}

void solve() {
    ll a, m;
    cin >> a >> m;
    ll inv, tmp;
    exgcd(a, m, inv, tmp);
    inv = (inv % m + m) % m;  // 处理负数
    cout << inv << '\n';
}
```
