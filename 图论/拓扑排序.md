# 拓扑排序

```cpp
queue<int> q;
vector<int> ans;
for (int i = 1; i <= n; i++) {
    if (p[i] == 0) q.push(i);
}

while (!q.empty()) {
    int t = q.front();
    q.pop();
    ans.push_back(t);
    for (auto x : mp[t]) {
        p[x]--;
        if (p[x] == 0) q.push(x);
    }
}
```

**算法说明**：
- `p[i]`：节点 i 的入度
- `mp[t]`：节点 t 的所有出边
- 每次将入度为 0 的节点加入队列

**适用场景**：
- 有向无环图（DAG）的拓扑排序
- 判断是否存在环（结果中节点数 < n 则存在环）
