# 字符串哈希

```cpp
const int mod = 1e9 + 7;
const int BASE = 131;
const int N = 1e6 + 5;
string str;
int n;
int Hash[N], P[N];

void initHASH() {
    P[0] = 1;
    for (int i = 1; i <= n; i++) {
        Hash[i] = (Hash[i - 1] * BASE + str[i]) % mod;
        P[i] = (long long)P[i - 1] * BASE % mod;
    }
}
```

**算法说明**：
- `Hash[i]`：字符串前 i 个字符的哈希值
- `P[i]`：BASE^i % mod，用于快速计算子串哈希
- `BASE = 131`：常用基数，经验值
- `mod = 1e9 + 7`：常用模数

**子串哈希计算**：
```cpp
// 计算子串 [l, r] 的哈希值
int getHash(int l, int r) {
    return (Hash[r] - (long long)Hash[l-1] * P[r-l+1] % mod + mod) % mod;
}
```

**常见模数组合**：
- (1e9+7, 131) 或 (1e9+9, 137)
