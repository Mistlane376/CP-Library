# 基础算法模板

## 二分查找模板

```cpp
int l = 0, r = INT_MAX;
while (l <= r) {
    int mid = (l + r) / 2;
    if (judge(mid)) {
        // 满足条件
        l = mid + 1;
    } else {
        // 不满足条件
        r = mid - 1;
    }
}
```

**适用场景**：
- 答案满足单调性
- 求最大/最小符合的值

## 三分查找模板

```cpp
double l = 0, r = 1e9;
while (r - l > 1e-9) {
    double m1 = (2 * l + r) / 3;
    double m2 = (l + 2 * r) / 3;
    if (f(m1) < f(m2))
        r = m2;
    else
        l = m1;
}
double ans = f(l);
```

## 滑动窗口（双端队列）

```cpp
deque<int> dq;
for (int i = 0; i < n; i++) {
    while (!dq.empty() && a[dq.back()] < a[i]) dq.pop_back();
    dq.push_back(i);
}
```
