# 位运算与格雷码

## 常用位运算技巧

### 判断奇偶

```cpp
if (n & 1) {
    // 奇数
}
```

### 清零第 k 位

```cpp
n & (~(1 << k))
```

### 其他常用操作

```cpp
// 设置第 k 位
n |= (1 << k)

// 切换第 k 位
n ^= (1 << k)

// 判断第 k 位是否为 1
if (n & (1 << k)) {
    // 第 k 位是 1
}

// 取最低位的 1
int lowbit = n & (-n)

// 统计二进制中 1 的个数
int count = __builtin_popcount(n)

// 消去最低位的 1
n &= (n - 1)
```

## 格雷码

格雷码是一种二进制编码，相邻两个数只有一位不同。

```cpp
// 生成 n 位格雷码
vector<int> grayCode(int n) {
    vector<int> res;
    for (int i = 0; i < (1 << n); i++) {
        res.push_back(i ^ (i >> 1));
    }
    return res;
}
```

**应用场景**：
- 减少编码错误
- 数字电路
- 算法竞赛中的状态转移
